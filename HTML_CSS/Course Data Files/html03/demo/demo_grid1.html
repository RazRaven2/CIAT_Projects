Grid Content Alignment

This demo explores the impact of the align-items and justify-items properties on the placement of grid content. The align-items property controls the placement of content along the column axis or vertically within each grid cell. The justify-items property positions content along the row axis or horizontally within each cell.

To explore these two properties, use the web form controls below and view their impact on the preview grid on the right.

Triad Colors

Triad colors are three hues separated by 90° on the color wheel. This color scheme provides the same opportunity for pleasing color contrasts as complementary color designs, but is not as striking. Select a color using the controls below and then rotate the hue value on the color wheel to view different sets of triad colors.

hsl(0, 100%, 50%) hsl(120, 100%, 50%) hsl(240, 100%, 50%)
rgb(255, 0, 0) rgb(0, 255, 0) rgb(0, 0, 255)
